# Multica

Open-source platform for assigning software issues to coding agents, tracking execution, and compounding reusable team skills.

## Audience

- Developers, AI-native engineering teams

## Canonical Links

- **Homepage:** https://multica.ai
- **GitHub:** https://github.com/multica-ai/multica
- **Changelog:** https://github.com/multica-ai/multica/releases

## Tags

- `agent`, `coding`, `orchestration`, `cli`, `task-management`

## Notes

Multica is built for teams that want Claude Code or Codex agents to operate like real collaborators rather than one-off prompt sessions. It combines a task board, local or cloud runtimes, agent profiles, CLI tooling, and reusable skills so teams can assign issues, monitor progress, and keep agent execution inside an ongoing engineering workflow.
