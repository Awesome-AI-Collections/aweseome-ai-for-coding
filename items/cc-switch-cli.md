# CC-Switch CLI

一个统一管理 Claude Code、Codex、Gemini、OpenCode 与 OpenClaw 配置、MCP、提示词和环境检查的跨平台 CLI 工具。

## 适合谁看

- 开发者、AI CLI 重度用户、工程团队

## 官方链接

- **GitHub:** https://github.com/SaladDay/cc-switch-cli
- **更新记录:** https://github.com/SaladDay/cc-switch-cli/blob/main/CHANGELOG.md

## 标签

- `Claude Code`, `Codex`, `Gemini`, `MCP`, `CLI`

## 说明

CC-Switch CLI 适合同时使用 Claude Code、Codex、Gemini、OpenCode 或 OpenClaw 的开发者。它把 provider 切换、临时启动、MCP 同步、提示词管理、技能同步、本地代理路由和环境检查集中到一个 CLI/TUI 工具里，减少手工改配置和多套命令来回切换的成本。根据 2026 年 4 月 3 日发布的 v5.3.0 说明，它已经支持在不改全局当前 provider 的前提下临时启动 Claude 或 Codex，并继续增强 MCP 环境变量编辑和多应用配置流。
